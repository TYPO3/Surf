=====================================
How to deploy other apps and websites
=====================================

There are several ways to deploy other projects with TYPO3 Surf. For example, you can:

* Provide a composer package with the application and tasks you need.
  Particularly useful for complex workflows.
* Use \TYPO3\Surf\Application\BaseApplication or \TYPO3\Surf\Domain\Model\Application
  in your recipe and enrich the workflow with necessary tasks.

.. seealso:: :doc:`How to create and add tasks <../../Tasks/Index>`.
